You are given a string with a long name which includes several words.
Each word should be separated by a single dash ("-") character. You should remove all extra dashes.

Input: A long name with several words, a string (Unicode for python2.7).

Output: A long name without extra dashes, a string.

Example: 

checkio('I---like--python') == "I-like-python"

How it is used: Some symbols when placed near each other look like one to the human eye, but this is not the case for computer programs. Because of this, string normalization is a significant part of OCR (optical character recognition) systems and is critical for speech synthesis software.
